Coding

  • TPIE - The Templated Portable I/O Environment November, 2013 - October, 2015

    TPIE is a library for manipulating large datasets on a desktop computer. TPIE provides implementations of several external memory algorithms and data structures as well as a framework for efficient and portable data processing. While working as a student programmer at MADALGO the majority of my time was spent working on TPIE. Some of my work is described here:

    • B-tree implementation in external memory and code for handlings blocks on disk. Based upon existing ab-tree code in internal memory.
    • Improvements to the pipelining framework by adding functionality to change the order in which pipeline phases are run(flush priorities) and limiting sending of out-of-band data.
    • Moving large parts of TPIE code to C++11 making use of C++ move semantics and perfect forwarding.
    • Organizing and writing documentation for TPIE.
    • Prototyping improvements on the multiway-merge sorting algorithm implemented in TPIE.

  • SCALGO SCANJanuary, 2016 - July, 2016

    While at SCALGO I worked on the product SCAN(SCALGO Combinatorial Anti Noise); a commercial product for cleaning massive sonar point clouds in external memory. My work consisted of reimplementing large parts of the code in the TPIE pipelining framework allowing for higher performance and easier maintenance of the code base.
